# Flappy Bird minisajt (GitHub Pages)

## Så publicerar du på GitHub Pages
1. Skapa ett repo och ladda upp: index.html, style.css, script.js
2. Gå till **Settings → Pages**
3. Under “Build and deployment”: välj **Deploy from a branch**
4. Välj branch: **main** och folder: **/(root)** → Save
5. Vänta tills du får en länk, t.ex. `https://dittnamn.github.io/flappy-site/`

## Bädda in i Google Sites
Google Sites: **Infoga → Bädda in → URL** och klistra in GitHub Pages-länken.
